== Description ==

With this plugin you can add 'WDV About Me' widget to your theme.

== Features ==

* Create 'WDV About Me' widget

WDV About Me Widget plugin is created by [wdvillage](https://wdvillage.com)


== Installation ==

1. Upload wdv-about-me-widget.zip throught tgm installition or upload unzipped folder to your wp-content/plugins/ folder
2. Activate the plugin through the "Plugins" menu in WordPress
