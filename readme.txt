Add 'WDV About Me' widget to your theme.

== Description ==

With this plugin you can add 'WDV About Me' widget to your theme. You can add you data or your firm data. You can add image, description, link, contact information, social links.

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/wdv-about-me-widget directory, or install the plugin through the WordPress plugins screen directly.
2. In your WordPress administration, go to the Plugins page.
3. Activate the 'WDV About Me Widget' plugin through the 'Plugins' screen in WordPress. 
1. Now you can add widget 'WDV About Me Widget'to your WP theme widget area.
